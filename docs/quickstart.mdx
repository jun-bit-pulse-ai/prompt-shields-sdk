# Quickstart

Get AI asset data from Prompt Shields into your system in three steps.

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log into Atlas AI Dashboard">
    Go to [app.promptshields.io/partners](https://app.promptshields.io/partners) and navigate to **Partners > Add Integration**.
  </Step>
  <Step title="Select your partner type">
    Choose your EA platform (Ardoq, LeanIX, ServiceNow, or Custom).
  </Step>
  <Step title="Copy credentials">
    You'll receive:
    - `client_id` — e.g., `ps-partner-ardoq-abc123`
    - `client_secret` — shown once, save securely
    - `api_key` — fallback key for simpler integrations
  </Step>
</Steps>

## Step 2: Authenticate

<Tabs>
  <Tab title="OAuth 2.0 (Recommended)">
    ```bash
    curl -X POST https://api.promptshields.io/oauth/token \
      -H "Content-Type: application/x-www-form-urlencoded" \
      -d "grant_type=client_credentials" \
      -d "client_id=ps-partner-ardoq-abc123" \
      -d "client_secret=sk-partner-secret-xyz789"
    ```

    Response:
    ```json
    {
      "access_token": "eyJhbGciOiJSUzI1NiIs...",
      "token_type": "Bearer",
      "expires_in": 3600,
      "scope": "registry:read data-flows:read risks:read"
    }
    ```
  </Tab>
  <Tab title="API Key (Fallback)">
    Skip the token exchange and use your API key directly:

    ```bash
    curl https://api.promptshields.io/partner/v1/assets \
      -H "Authorization: Bearer ps-pk-your-key-here"
    ```
  </Tab>
</Tabs>

## Step 3: Fetch AI Assets

```bash
curl https://api.promptshields.io/partner/v1/assets \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiIs..."
```
