# Python SDK

The Prompt Shields Python SDK is a drop-in replacement for the OpenAI and Anthropic clients. Every LLM call is wrapped with structured telemetry — fail-open, so a collector outage never blocks a model call.

<Info>
  For a deep-dive walkthrough with debugging tips and production patterns,
  see the standalone [SDK Developer Guide](https://github.com/jun-bit-pulse-ai/prompt-shields-sdk/blob/main/SDK_GUIDE.md).
</Info>

## Installation

```bash
# OpenAI only
pip install "prompt-shields[openai]"

# Anthropic only
pip install "prompt-shields[anthropic]"

# Both providers
pip install "prompt-shields[all]"
```

Requires Python 3.11+.

## Quick Start

<Tabs>
  <Tab title="OpenAI">
    ```python
    from prompt_shields import ShieldsOpenAI

    client = ShieldsOpenAI(
        api_key="sk-...",
        ps_api_key="ps-...",

        business_unit="HR",

        owner="jane.doe@acme.com",
        data_classification="confidential",

    )
    ```

## Per-Request Metadata

For data that varies per call, use `ps_metadata`:

```python
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Review this candidate file..."}],
    ps_metadata={
        "data_sources": ["candidates_db", "linkedin_api"],
        "output_destination": "hiring_dashboard",
        "risk_tags": ["pii", "gdpr"],
        "session_id": "review-2025-04-12-001",
        "user_id": "hr_user_42",
    },
)
```

| Field | Purpose |
|-------|---------|
| `data_sources` | Systems feeding data into the prompt — populates data flow lineage |
| `output_destination` | Where AI output goes — completes the lineage graph |
| `risk_tags` | Free-form tags for governance filtering (`gdpr`, `hipaa`, ...) |
| `session_id` | Groups related calls in a multi-turn conversation |
| `user_id` | Opaque identifier (hash before passing if you need to anonymize) |

## Configuration Reference

| Argument | Type | Default | Notes |
|----------|------|---------|-------|
| `api_key` | str | — | LLM provider key. Never sent — only a SHA-256 fingerprint. |
| `ps_api_key` | str | — | Prompt Shields tenant key. |
| `vendor` | str | `"openai"` | Or `"anthropic"`. Set by typed subclasses. |
| `ps_collector_url` | str | `http://localhost:8000` | Set to your production collector URL. |
| `business_unit` | str | — | Becomes part of the asset merge key. |
| `use_case` | str | — | Maps to `use_case_name` in the registry. |
| `owner` | str | — | Maps to `owner_email`. |
| `data_classification` | str | — | `public` / `internal` / `confidential` / `restricted`. |
| `environment` | str | — | Separate envs are separate assets. |
| `calling_service` | str | — | Service/app name for dedup fallback. |
| `scan_pii` | bool | `True` | Disable to skip local PII detection. |
| `send_prompt_text` | bool | `False` | Opt-in to log full prompt content. |
| `pricing_table` | dict | built-in | Override token-to-USD pricing. |

## Privacy Model

| Field | Sent by default? |
|-------|------------------|
| Vendor + model | Yes |
| Token counts, latency, cost | Yes |
| Tool call names | Yes |
| PII category labels | Yes |
| API key fingerprint | Yes (SHA-256, truncated) |
| Business metadata | Yes (only what you pass) |
| **Prompt content** | **No** (unless `send_prompt_text=True`) |
| **Response content** | **Never** |
| **Raw API key** | **Never** |
| **Tool call arguments** | **Never** |

## PII Detection

The SDK runs a local pattern-based scanner before sending telemetry:

```python
from prompt_shields import detect_pii_categories

detect_pii_categories("Email jane@acme.com about SSN 123-45-6789")
# → ['email', 'ssn']
```

Detected categories: `email`, `phone`, `ssn`, `credit_card`, `ip_address`, `iban`, `health_data`, `financial_data`.

## Cost Estimation

Built-in pricing table for OpenAI, Anthropic, and Google models:

```python
from prompt_shields import estimate_cost

estimate_cost("openai", "gpt-4o", tokens_in=1000, tokens_out=2000)
# → 0.0225 (USD)
```

Override with a custom table:

```python
client = ShieldsOpenAI(
    api_key="sk-...",
    ps_api_key="ps-...",
    pricing_table={
        ("openai", "gpt-4o"): (0.0020, 0.0080),  # negotiated rate
        ("custom", "internal-llm-v1"): (0.0001, 0.0002),
    },
)
```

## Fail-Open Guarantees

1. **LLM calls never blocked by telemetry** — response returned before telemetry sent
2. **No prompt content leaves the host** unless explicitly opted in
3. **Telemetry errors never raise** — logged at WARN, buffered for retry
4. **Buffer overflow drops oldest events** — never blocks the app
